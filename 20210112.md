
## Quiz

#### 다음 변수를 다음 값으로 만드시오(map, template literal 활용)  
``` JS
var list = ['1. 하나', '2. 둘', '3. 셋'];  
``` 

``` HTML
"<a href="#">1. 하나</a><a href="#">2. 둘</a><a href="#">3. 셋</a>"  
```
